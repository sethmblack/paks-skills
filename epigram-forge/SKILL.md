# Epigram Forge

Compress complex observations into single memorable sentences with the structure and elegance of Oscar Wilde's epigrams.

---

## When to Use

- Creating a tagline or headline
- Summarizing an argument memorably
- Making content quotable
- Writing a speech that will be remembered
- User asks "make this quotable" or "give me a one-liner"

---

## Inputs

| Input | Required | Description |
|-------|----------|-------------|
| observation | Yes | The insight, idea, or observation to be compressed |
| context | No | Where this will be used (speech, marketing, social media, etc.) |
| tone | No | Cynical, warm, philosophical, subversive |

---

## Workflow

### Step 1: Gather and Review Inputs

Review all provided information and context:
- Examine the inputs provided
- Identify key parameters and constraints
- Clarify any ambiguities

### Step 2: Apply the Framework

Execute the core methodology:
- Follow the analysis approach outlined above
- Document findings systematically
- Consider multiple perspectives

### Step 3: Generate Outputs

Create the final deliverable:
- Synthesize insights from analysis
- Format according to output specifications
- Ensure actionability of recommendations

## Crafting Checklist

For any epigram, verify:

- [ ] **Compression** - Can any word be cut without losing meaning?
- [ ] **Surprise** - Does it say something unexpected?
- [ ] **Rhythm** - Does it please the ear when spoken aloud?
- [ ] **Standalone** - Does it make sense without context?
- [ ] **Universal** - Does it apply beyond the immediate situation?
- [ ] **Quotable** - Would someone want to repeat this?

---

## Constraints

- One sentence maximum (two short sentences acceptable for doubled definition pattern)
- Every word must earn its place
- Avoid adjectives that could be cut
- The insight must be genuinely true, not merely clever
- Test by reading aloud--it must sound right
- If it requires explanation, it has failed

## Epigram Forge

### The Core Insight
Self-consciousness assumes we matter more to others than we do. Our anxiety about others' judgment is ironically self-centered.

### The Epigram
"The only thing worse than being talked about is not being talked about."

### Structural Analysis
- Pattern used: Balanced antithesis (two phrases in opposition)
- Key technique: Inverts the expected value (being talked about = bad â†’ good)
- Rhythm notes: Parallel structure creates satisfying symmetry; "worse" does double duty (literal and ironic)

### Variations
1. "We would worry less about what others think of us if we realized how seldom they do." (The Turn pattern - setup/pivot)
2. "Obscurity is the punishment for imagining we matter." (The Twist pattern - builds, then final word reframes)
3. "Everyone is the center of their own universe, which explains why collision is so rare." (Paradox Statement - seems contradictory but true)

### Application Notes
Use the first version for contexts about reputation, publicity, or social anxiety. The second works for humility contexts. The third fits discussions of loneliness or disconnection. All compress the same insight with different emphases.
